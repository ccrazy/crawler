- [comment_data.sql](http://ojxezaira.bkt.clouddn.com/comment_data.sql)为一周热销产品的评论信息

- [master_comment_data.sql](http://ojxezaira.bkt.clouddn.com/master_comment_data.sql)为所有产品的评论信息
